Questão 1: Troca de valores entre duas variáveis
TrocaValores.java: Código fonte
variaveis.txt: Arquivo de entrada com os valores das variáveis
saida.txt: Arquivo de saída gerado pelo programa

Questão 2: Cálculo de salário dos vendedores
CalculoSalario.java: Código fonte
vendedores.txt: Arquivo de entrada com os dados dos vendedores
saida.txt: Arquivo de saída gerado pelo programa

Questão 3: Cálculo da compra de camisas
CalculoCamisas.java: Código fonte
camisas.txt: Arquivo de entrada com os dados das compras de camisas
saida.txt: Arquivo de saída gerado pelo programa

Questão 4: Cálculo da duração de jogos de xadrez
CalculoXadrez.java: Código fonte
xadrez.txt: Arquivo de entrada com os dados dos jogos de xadrez
saida.txt: Arquivo de saída gerado pelo programa

Baixe todos os arquivos e os insira no mesmo repositório.

Para compilar e executar os programas, certifique-se de ter instalado em seu sistema:
- Java Development Kit (JDK)
- Um editor de texto ou IDE (como VS Code, IntelliJ IDEA, Eclipse)
- XAMPP (opcional, como servidor)

Instruções de Compilação pelo prompt de comando:
1. Navegue até o diretório do projeto;
2. Compile o arquivo;
3. Execute o programa desejado;
4. Verifique o arquivo `saida.txt` para visualizar os resultados gerados.
